Данный проект содержит модуль с функциями для построения характеристик радиолокационных станций (РЛС) по заданным параметрам.
Проект может применяться как для обучения основам радиолокации в области первичной обработки сигналов, так и для работы по данной тематике.
Проект состоит из:
1. radar - главный модуль, содержащий функции:
	1.1 - Построение диаграммы направленности (ДН) антенной решетки (АР) в азимутальной плоскости.
	1.2 - Построение диаграммы направленности антенной решетки в угломестной плоскости.
	1.3 - Расчет ширины диаграммы направленности по уровню половинной мощности.
	1.4 - Построение огибающих откликов пространственных фильтров в угломестной плоскости.
	1.5 - Пеленгационная характеристика методом сканирования пространства.
	1.6 - Формирование таблицы отношений методом отношений максимальных амплитуд в паре лучей.
	1.7 - Пеленгационная характеристика методом отношений максимальных амплитуд в паре лучей.
	1.8 - Формирование таблицы отношений методом отношений максимальной амплитуды в луче к максимальной амплитуде в соседнем луче.
	1.9 - Пеленгационная характеристика методом отношений максимальной амплитуды в луче к максимальной амплитуде в соседнем луче.
	1.10 - Проводка цели по высоте.
	1.11 - Построение зоны обнаружения по исходным данным.
	1.12 - Построение зоны обнаружения по файлу регистрации.
	1.13 - Построение зависимостей отношения сигнал/шум (ОСШ) от дальности до цели.
	1.14 - Построение зависимостей потенциальной ошибки по угловой координате от дальности до цели.
	1.15 - Вспомогательные функции для расчетов характеристик, приведенных выше.
2. dn_beta - пример расчета и построения ДН АР в азимутальной плоскости.
3. dn_eps - пример расчета и построения ДН АР в угломестной плоскости.
4. peleng_scan - пример расчета и построения пеленгационной характеристики методом сканирования.
5. provodka - пример расчета и построения проводки цели по высоте.
6. far_zone_no_input_data - пример расчета и построения зоны обнаружения по исходным данным.
7. far_zone_with_input_data - пример построения зоны обнаружения по файлу регистрации, а также зависимостей ОСШ и потенциальной ошибки по угловым координатам от дальности до цели.
8. ratio_table_pairs - пример формирования таблицы отношений методом отношений максимальных амплитуд в паре лучей.
9. peleng_pairs - пример построения пеленгационной характеристики методом отношений максимальных амплитуд в паре лучей.
10. ratio_table_neighbor - пример формирования таблицы отношений методом отношений максимальной амплитуды в луче к максимальной амплитуде в соседнем луче.
11. peleng_neighbor - пример построения пеленгационной характеристики методом отношений максимальной амплитуды в луче к максимальной амплитуде в соседнем луче.

Параметры РЛС, приведенные в примерах, взяты лишь для демонстрации и не являются параметрами настоящих РЛС.
Проект написан на Python версии 3.7. Для работы с проектом необходимо установить библиотеки numpy и matplotlib.
